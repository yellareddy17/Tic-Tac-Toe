# Tic-Tac-Toe

A simple Tic-Tac-Toe game implemented in JavaScript, HTML, and CSS.

## Features
- Two-player mode
- Simple and intuitive UI
- Displays game status (winner or draw)
- Restart button to play again

## Installation

1. Clone the repository:
   ```sh
   git clone https://github.com/yellareddykalluru/tic-tac-toe.git
   ```
2. Navigate to the project folder:
   ```sh
   cd tic-tac-toe
   ```
3. Open `index.html` in a web browser.

## How to Play
1. The game starts with player 'X'.
2. Players take turns clicking on empty squares to mark their symbol ('X' or 'O').
3. The game announces a winner when three marks align horizontally, vertically, or diagonally.
4. If all squares are filled and no player has won, the game ends in a draw.
5. Click the restart button to play again.
